# Audio Ambiental

Coloca aquí el audio ambiental para el sitio web.

**Archivo necesario:**
- ambient.mp3

**Recomendaciones:**
- Duración: 1-3 minutos (se reproduce en loop)
- Formato: MP3
- Bitrate: 128kbps
- Peso: < 5MB
- Contenido: Sonidos suaves de cafetería (café moliendo, conversaciones suaves, fondo musical)

**Fuentes de audio sugeridas:**
- Freesound.org (audio libre)
- Epidemic Sound
- AudioJungle
- Artlist

**Nota:**
El audio es completamente opcional. Si no colocas ningún archivo,
el controlador de audio simplemente no funcionará y no afectará
la experiencia del sitio.
