# Video de Ubicación

Coloca aquí el video de tu ubicación en Google Maps.

**Archivo necesario:**
- location.mp4

**Opcional:**
- location-poster.jpg (imagen de portada del video)

**Recomendaciones para el video:**
- Duración: 10-30 segundos
- Resolución: 1280x720px (HD)
- Formato: MP4 (H.264)
- Peso: < 10MB
- Contenido: Screen recording de Google Maps mostrando tu ubicación

**Cómo crear el video:**
1. Abre Google Maps
2. Busca tu ubicación
3. Graba la pantalla mostrando la ubicación
4. Exporta como MP4
5. Comprime si es necesario
